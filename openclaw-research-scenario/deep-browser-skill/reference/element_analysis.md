# 元素分析参考文档

本文档提供页面元素分析方法论，帮助识别交互元素、判断安全性、评估导航价值。

## 分析目标

1. **交互类型** — 元素如何被操作
2. **安全性判断** — 只读还是写入（关键）
3. **导航价值** — 操作是否能发现新页面
4. **match_level** — 元素引用是否可靠

## 快速识别

`opencli browser state` 返回的快照中，每个交互元素带 `[N]` 数字引用：

```
[1] link "商品列表"
[2] button "展开筛选"
[3] textbox "搜索"
[4] button "搜索"
[5] button "提交订单"
```

## 只读/写入分类（关键维度）

### 只读类（允许操作）

| 元素特征 | 交互结果 | 示例 |
|---------|---------|------|
| 导航链接 | 跳转到新页面 | `[1] link "商品列表"` |
| 展开按钮 | 显示隐藏内容 | `[2] button "展开筛选"` |
| 搜索框+按钮 | 展示搜索结果 | `[3]+[4]` |
| 分页控件 | 加载其他页数据 | `[6] link "下一页"` |
| 筛选控件 | 改变内容展示 | `[7] select "分类"` |
| 滚动触发 | 加载更多内容 | state 提示 has_more_below |

### 写入类（禁止操作）

| 元素特征 | 交互结果 | 示例 |
|---------|---------|------|
| 注册提交 | 创建新账号 | `button "注册"` |
| 登录提交 | 创建用户会话 | `button "登录"` |
| 联系表单提交 | 发送消息 | `button "发送"` |
| 购买/下单 | 产生交易 | `[5] button "提交订单"` |
| 编辑提交 | 修改已有数据 | `button "保存"` |
| 删除操作 | 删除数据 | `button "删除"` |

**核心判断**：操作目的是获取/展示信息 → 只读 → ✅；操作目的是创建/修改数据 → 写入 → ❌。

**模糊判断**：无法确定时视为写入类，禁止操作。

## 导航价值评估

| 元素类型 | 导航价值 | 说明 |
|---------|---------|------|
| 导航链接 | ⭐⭐⭐ | 直接发现新页面 URL |
| 面包屑 | ⭐⭐⭐ | 层级导航 |
| 分页控件 | ⭐⭐⭐ | 更多列表条目 |
| 搜索结果 | ⭐⭐⭐ | 搜索后发现的链接 |
| 折叠菜单 | ⭐⭐ | 展开后发现的链接 |
| 标签切换 | ⭐⭐ | 切换后发现的链接 |
| 筛选/排序 | ⭐ | 改变展示但不产生新 URL |

## match_level 容错

操作返回的 `match_level` 指示元素引用可靠性：

| level | 含义 | 处理 |
|-------|------|------|
| `exact` | 完全匹配 | 直接继续 |
| `stable` | 软属性漂移 | 操作仍有效，建议验证结果 |
| `reidentified` | 原引用消失，找到替代 | 需要二次确认 |

**规则**：`reidentified` 时应重新 `opencli browser state` 获取新快照。

## Compound 信息

state 快照的 `compounds` sidecar 提供结构化表单控件信息：

**日期控件**：`{ "control": "date", "format": "YYYY-MM-DD" }`
**选择控件**：`{ "control": "select", "options": [...] }`
**文件控件**：`{ "control": "file", "accept": "image/*" }`

不要用正则猜测格式，直接用 compound 信息。

## 操作建议格式

```markdown
## 建议操作

1. ✅ **click** (只读 - 导航)
   - 命令: opencli browser click 1
   - 目的: 进入商品列表页
   - 导航价值: ⭐⭐⭐

2. ✅ **click** (只读 - 展示)
   - 命令: opencli browser click 2
   - 目的: 展开筛选面板
   - 导航价值: ⭐⭐

3. ✅ **type + keys** (只读 - 搜索)
   - 命令: opencli browser type 3 "关键词" && opencli browser keys Enter
   - 目的: 搜索
   - 导航价值: ⭐⭐⭐

4. ❌ **click** (写入 - 禁止)
   - 目标: [5] 提交订单
   - 原因: 会产生交易
   - 处理: 记录入口，不执行
```
